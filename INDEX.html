🚀 Getting Started

⏱️ Total Setup Time: 6 minutes

Step 1: Supabase Setup (2 min)

  1. Go to https://supabase.com
  2. Create FREE account and new project
  3. Copy Project URL and API Key
  4. Run database/schema.sql in SQL Editor

Step 2: Configure Backend (1 min)

  1. Open backend/app.py
  2. Update SUPABASE_URL and SUPABASE_KEY
  3. Save file

Step 3: Start Backend (2 min)

Windows: Double-click run_backend.bat
Mac/Linux: chmod +x run_backend.sh && ./run_backend.sh
Manual: cd backend && pip install -r requirements.txt && python app.py

Step 4: Open Frontend (1 min)

Open frontend/index.html in your browser

✅ That's it! Your complaint system is now running.
